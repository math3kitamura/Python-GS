## Descrição do Projeto

O Prevensor de Queimadas Naturais é um sistema desenvolvido para analisar dados ambientais e estimar o risco de queimadas em uma região.
O projeto utiliza informações inseridas pelo usuário e aplica uma lógica de cálculo e classificação, produzindo um resultado em forma de DataFrame.

### As variáveis analisadas são:

- Temperatura (°C)

- Umidade relativa (%)

- Velocidade do vento (km/h)

- Dias sem chuva

### Com base nesses dados, o sistema informa se o risco é:

- Baixo Risco

- Médio Risco

- Alto Risco

## Relevância Ambiental

As queimadas constituem um dos maiores desafios ecológicos da atualidade. Seus impactos incluem:

- perda de biodiversidade;

- piora da qualidade do ar;

- danos à saúde humana;

- influência negativa no clima;

- destruição de áreas agrícolas e ecossistemas.

A identificação precoce de condições favoráveis ao fogo é fundamental para a prevenção.
Este projeto demonstra, em formato acadêmico, como dados ambientais simples podem ser transformados em indicadores de risco, auxiliando na conscientização e no planejamento preventivo.

## Funcionamento do Sistema

### O programa foi estruturado para:

- Solicitar ao usuário os dados ambientais.

- Processar essas informações por meio de uma classe específica que calcula a probabilidade de risco.

- Classificar o risco com base nos valores fornecidos.

- Exibir todos os dados organizados em um DataFrame.

### O código utiliza:

- programação orientada a objetos (POO);

- funções auxiliares para coleta e validação de respostas;

- uma função encadeada responsável por montar o DataFrame final.

## Objetivo Acadêmico

Este projeto foi criado com finalidade educacional, integrando:

- conceitos de programação;

- análise ambiental básica;

- boas práticas de organização e modularização de código;

- construção de sistemas interativos simples.

Ele demonstra como a tecnologia pode ser utilizada como ferramenta de apoio à prevenção ambiental.

## Conclusão

O Prevensor de Queimadas Naturais mostra como dados ambientais podem ser utilizados para identificar condições de risco, promovendo conscientização e reforçando a importância de iniciativas voltadas à preservação do meio ambiente.
